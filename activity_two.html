    2. The Document Object Model, or DOM, is a tree like structure that depicts the relationship between all of the elements on a webpage.

  2. What is a container and what are they used for?

    3. A container is meant to provide all sub-elements within the container with common attributes, such as alignment, padding, and text size.

  3. What is semantic HTML and why is it important?

    4. Semantic HTML is meant to clearly represent the purpose of elements within the code. This can be done utilizing semantic tags such as <header>. This is important for the sake of readability and documentation. It makes it much clearer what each element presented does, and where it belongs.
